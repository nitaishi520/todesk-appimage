## 🚀 一键使用
```bash

wget https://github.com/nitaishi520/todesk-appimage/releases/download/v4.8.5.1/ToDesk-v4.8.5.1-amd64.AppImage

# 运行
chmod +x ToDesk-*.AppImage
./ToDesk-*.AppImage
```
